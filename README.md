# village_services

Village Services — A Smart Rural Service Coordination Platform.

## 🚀 Features
- Tractor & equipment booking
- Labor hiring system
- Village complaint system
- Driver work logging
- Veterinary appointment booking

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript (React + Vite)
- Backend: Node.js
- Database: MongoDB

## 📌 Purpose
This project aims to digitize essential rural services into one unified web platform, making them more accessible and efficient.
